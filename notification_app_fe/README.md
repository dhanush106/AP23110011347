# Notification App Frontend

This is the frontend component of the Notification Management System, built with React and Vite.

## 🚀 Features

- **Responsive Dashboard**: Works on all screen sizes.
- **Search & Filter**: Find notifications quickly.
- **Priority Toggle**: Switch between all notifications and priority updates.
- **API Integration**: Fetches data from a secure REST API.
